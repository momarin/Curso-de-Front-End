# Formulário baseado no estilo dos Google Forms

# Projeto para consolidação de conhecimentos de HTML e CSS

# O que chama atenção:
    - evento de hover com transition;
    - separação de <ul> em duas caixas inline;
    - seleção do tipo de arquivo e permissão para up de mais de um arquivo no input type="file";
    - uso do span para alteração das cores dos astericos de required field;
    - responsividade com width 100% nos inputs, fazendo com que eles nunca ultrapassem o form-body;

# Diferenças do projeto original
    - alteração de fonte;
    - responsividade;
    - alteração de vários títulos;
